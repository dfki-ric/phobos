phobos.blender.model.geometries.deriveScale(obj)¶

Returns the scale of the specified object.

Object scale is gathered from the matrix_world, as the link scales in Blender might change the mesh scale, too.

Parameters:

obj (bpy.types.Object) – object to derive the scale of

Returns:

three scale floats (x, y, z) combined from all parents and the object itself

Return type:

list
